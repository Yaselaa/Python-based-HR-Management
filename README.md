## 🇺🇸 English: Project Overview
This project is a Python-based **HR Management Dashboard** that consolidates various HR operational tools into a single, professional interface. It acts as a centralized launcher for recruitment, leave management, and data analysis modules.

### ✨ Key Features
* **Centralized Dashboard:** A unified hub to launch multiple HR software modules instantly.
* **Modern UI/UX:** Clean, corporate-themed interface optimized with Tkinter and Pillow.
* **Sleek Design:** Features a custom sidebar, hover-responsive buttons, and professional visual separators.
* **Standalone Executable:** Fully compatible with PyInstaller for distribution as a single `.exe` file.
* **Privacy Focused:** The codebase is designed with a generic structure to ensure data privacy and corporate confidentiality.

### 🛠 Tech Stack
* **Language:** Python 3.12+
* **Libraries:** Tkinter, Pillow, Subprocess, PyInstaller.
* **Color Palette:** Corporate Deep Red (#913619) and Dark Blue (#2C3E50).
